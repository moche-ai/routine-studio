# routine-studio Agents

## Project-Specific Agents

### Video Producer Agent
Specialized in Remotion video generation workflows.
- Template selection
- Asset optimization
- Render queue management

### Channel Manager Agent  
Manages YouTube channel automation.
- Channel settings
- Branding assets
- Publishing schedules

### Content Writer Agent
Creates marketing content.
- Script generation
- SEO optimization
- Multi-language support

## Agent Workflows

### Video Creation Workflow
1. **Content Writer** generates script
2. **Video Producer** creates video
3. **Channel Manager** schedules publication

### Channel Setup Workflow
1. **Channel Manager** configures settings
2. **Video Producer** creates branding assets
3. **Content Writer** prepares descriptions

## Usage
Agents are invoked via the API or through the studio interface.
Configuration is stored in `agents/` directory.
